# WoWHead-Scraper
Scrapes WoWHead Transmogs and Outfits with php 

Supports all languages.  Usage

scrape.php?url=http://www.wowhead.com/outfit=23887/black-shattered-sun
pseudo usage
scrape.php?url=<wowheadlink>
Example usage transmog above is outfits. 
scrape.php?url=http://www.wowhead.com/transmog-set=1315/abandoned-fealty-battleplate
